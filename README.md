# heart-disease-prediction-ml
# Heart Disease Prediction using Machine Learning

##  Overview
This project focuses on predicting heart disease using Machine Learning techniques based on clinical and demographic data. The goal is to support early detection and improve healthcare decision-making.

---

## Dataset
- 303 patient records
- 14 features (age, cholesterol, chest pain, etc.)
- Source: Kaggle (Heart Disease Dataset)

---

## Machine Learning Models Used
- Linear Regression  
- Decision Tree  
- Random Forest  

---

## ⚙️ Approach
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Engineering  
- Model Training & Evaluation  
- 7-Fold Cross Validation  

---

##  Results
- Random Forest achieved ~99% accuracy  
- Decision Tree performed closely  
- Linear Regression showed lower performance  

---

## Key Insights
- Chest pain type, slope, and thal were important features  
- Feature importance helped in model interpretability  
- Ensemble methods improved performance
